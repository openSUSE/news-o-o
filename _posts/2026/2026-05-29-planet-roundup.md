This is a roundup of articles from the openSUSE community listed on [planet.opensuse.org](https://planet.opensuse.org).


The community blog aggregates a list of the featured highlights below from May 22 to 27.


Blogs this week cover security vulnerabilities discovered and patched in qSnapper's privileged D-Bus service, a new GSoC 2026 contributor joining the [openSUSE Project](https://www.opensuse.org/), nightly syslog-ng container images now available based on Alma Linux, a new plasmoid Scrolling Clock for KDE Plasma 6, a tip for previewing Markdown in the Kate editor, the April 2026 Krita report, the Mobile Linux Hackday in České Budějovice, [Agama 21](https://agama-project.github.io/blog/2026/05/21/agama-21) and more.


**Here is a summary and links for each post:**

## [Whaleshark Opens Knopje](https://vizZzion.org/blog/2026/05/whaleshark-opens-knopje/)

[Sébas](https://vizZzion.org/) announces that on June 19 he will open the next edition of Knopje with a 1.5-hour melodic techno set. 


## [Krita April 2026 Report](https://www.kdeblog.com/informe-de-abril-de-2026-de-krita.html)

The [KDE Blog](https://www.kdeblog.com/) covers the April 2026 Krita monthly report, which announces the release of Krita 5.3.2 and 6.0.2 with two months' worth of bug fixes and improvements including text tool enhancements, performance fixes, and an Android crash fix. The post also highlights upcoming features in development and improves wide-gamut color conversion.


## [Introducing Shared Canned Responses](https://openbuildservice.org/2026/05/28/request-workflow-improvements/)

The [Open Build Service Blog](https://openbuildservice.org/) introduces shared canned responses in OBS, expanding a feature that previously only allowed users to create personal canned responses under their own profiles. The update allows canned responses to now be shared across projects and packages, streamlining collaboration and communication in request workflows. 


## [How to Install exeLearning 4.0 on Your Computer](https://www.kdeblog.com/como-instalar-exelearning-4-0-en-tu-ordenador.html)

The [KDE Blog](https://www.kdeblog.com/) explains how to install exeLearning 4.0, a free and open-source tool for creating interactive digital educational resources. The post outlines the three available editions.


## [Scrolling Clock Widget: Plasmoids for Plasma 6 (29)](https://www.kdeblog.com/reloj-original-para-tu-escritorio-con-scrolling-clock-plasmoides-para-plasma-6-29.html)

The [KDE Blog](https://www.kdeblog.com/) presents Scrolling Clock, the 29th widget in their Plasma 6 plasmoid series, which displays an animated clock cycling through all digits for a unique and eye-catching desktop look. Users who enjoy the widget are encouraged to support the developer through ratings, comments, or donations on the KDE Store.


## [Previewing Markdown Files in the Kate Editor](https://victorhckinthefreeworld.com/2026/05/26/presvisualizar-archivos-markdown-en-el-editor-kate-de-kde/)

[Victorhck](https://victorhckinthefreeworld.com/) shares a quick tip for enabling live Markdown preview inside the KDE Kate editor by activating the "Document Preview" plugin. The author notes that HTML preview does not work the same way and recommends using a browser for that format instead.


## [Nightly syslog-ng Containers Based on Alma Linux](https://peter.czanik.hu/other/syslog-ng-nightly-containers-based-on-alma-linux/)

[Peter Czanik](https://peter.czanik.hu/) announces that nightly syslog-ng container images based on Alma Linux are now available on Docker Hub. Previously only Debian-based images were provided, but this new Alma Linux offering is built from the latest syslog-ng git snapshot packages. 


## [Accepted into Google Summer of Code 2026 with openSUSE!](https://mmarhin.github.io/gsoc2026blog/gsoc/opensuse/2026/05/26/accepted-into-gsoc.html)

[Mario Marín](https://mmarhin.github.io/) announces on his [GSoC 2026 blog](https://mmarhin.github.io/gsoc2026blog/) that he has been accepted into Google Summer of Code 2026 to contribute to the openSUSE project under two mentors. Over 12 weeks, he will work on improving the `obs-status-service`, including better SVG visualizations, a Gitea bot for Pull Request build information, and an AI-assisted stretch goal using Log Detective to analyze failed builds. He will be posting weekly progress updates on [his blog](https://mmarhin.github.io/gsoc2026blog/).


## [qSnapper: Various Security Issues in Privileged D-Bus Service (CVE-2026-41045 through CVE-2026-41049)](https://security.opensuse.org/2026/05/26/qsnapper-dbus-issues.html)

The [SUSE Security Team blog](https://security.opensuse.org/) discloses five CVEs found during a security review of qSnapper, a GUI frontend for the Btrfs snapshot manager snapper. All issues were addressed through coordinated disclosure with the upstream author and fixes were shipped in the qSnapper 1.3.3 released on May 26. 


## [MobileLinux Hackday #1 in České Budějovice](https://www.suse.com/c/first-mobilelinux-hackday-in-ceske-budejovice-outperforms-prague/)

The [SUSE Community Blog](https://www.suse.com/c/) reports that the first Mobile Linux Hackday held in České Budějovice had a bigger turnout than the well-established Prague series with regard to attendance and synergy. The Prague series had already built a strong following over seven monthly events and the new venue signals growing momentum and geographic expansion for the Mobile Linux hackday movement in Czechia.


## [How to Use Desktop Icons NG (DING) on openSUSE 16 GNOME](https://blog.geeko.jp/ribbon/3625)

The [Geeko Blog](https://blog.geeko.jp/) provides a fix for the Desktop Icons NG (DING) GNOME extension failing to work on openSUSE 16. The post includes the relevant error message from `/var/log/messages` to help users identify the problem.


## [Long-Term Support Doesn't Mean What You Think](https://www.kdeblog.com/soporte-a-largo-plazo-no-significa-lo-que-crees.html)

The [KDE Blog](https://www.kdeblog.com/) summarizes a post by KDE developer Nate Graham clarifying that LTS releases promise extended maintenance and security patches, not bug-free software or guaranteed personal support. The post draws a clear distinction between free community LTS distributions and commercially supported products and suggests that Flatpak apps can help bridge the software freshness gap on stable systems. 


## [Haruna 1.8 Released – New Version of This KDE Media Player](https://www.kdeblog.com/lanzado-haruna-1-8-nueva-version-de-este-reproductor-multimedia-de-kde.html)

The [KDE Blog](https://www.kdeblog.com/) announces the release of Haruna 1.8, an open-source video player built on libmpv with YouTube integration. Haruna continues to be a solid alternative to Dragon Player and Kaffeine within the KDE ecosystem.


## [Linux Saloon 202 | Early Edition](https://cubiclenate.com/2026/05/23/linux-saloon-202-early-edition/)

[CubicleNate](https://cubiclenate.com/) recaps episode 202 of the Linux Saloon podcast, covering Colin's use of his Surface Go running Cosmic Desktop, the release of Ubuntu 26.04 LTS, and updates on the Framework Computer Laptop 13 Pro. 


## [Linux Saloon 203 | News Flight Night](https://cubiclenate.com/2026/05/23/linux-saloon-203-news-flight-night/)

[CubicleNate](https://cubiclenate.com/) recaps episode 203 of the Linux Saloon podcast, during which attendees discuss Collin's experience with stillOS, the value of operating system rollback features, and notable news including HP sponsoring the Linux Vendor Firmware Service and KDE receiving a significant investment.


## [Xe Driver Support and Discover Improvements – This Week in Plasma](https://www.kdeblog.com/soporte-para-el-controlador-xe-y-mejoras-en-discover-esta-semana-en-plasma.html)

The [KDE Blog](https://www.kdeblog.com/) translates Nate Graham's weekly Plasma development summary. A standout community contribution added support for monitoring modern Intel Xe GPUs in System Monitor and its widgets. Discover also received several improvements including safer Flatpak data deletion sending files to the trash, a reorganized front page with the Editor's Choice section moved higher, and case-insensitive search on the Updates page.


## [Agama 21 Released](https://victorhckinthefreeworld.com/2026/05/22/publicado-agama-21/)

[Victorhck](https://victorhckinthefreeworld.com/) summarizes the release of Agama 21. The network configuration interface was redesigned with a new form supporting bond and bridge connections in addition to Ethernet and Wi-Fi. A new boot option `inst.remote=0` allows disabling remote installer access for improved security in sensitive environments.


## Tumbleweed – Review of the Weeks 2026/21

[Victorhck](https://victorhckinthefreeworld.com/2026/05/22/opensuse-tumbleweed-revision-de-la-semana-21-de-2026/) and [Dominique Leuenberger's blog](https://dominique.leuenberger.net/blog/2026/05/tumbleweed-review-of-the-week-2026-21/) recap week 21 with six snapshots. The releases shipped notable updates including AppArmor 5.0.0, KDE Plasma 6.6.5, Linux kernel 7.0.6 through 7.0.9, GStreamer 1.28.3, Ruby 4.0.4, and PostgreSQL 18.4. Upcoming pipeline changes include Agama 21, GCC 16 as the default system compiler, and a rework of Python 3 packaging.


## [Workshop: Agentic AI and Total Automation at Linux Center València](https://www.kdeblog.com/workshop-ia-agentica-y-automatizacion-total-en-linux-center-valencia.html)

The [KDE Blog](https://www.kdeblog.com/) announces a hands-on workshop on agentic AI and automation taking place on June 6 at Slimbook's Linux Center in Paterna, Valencia. The event features sessions on building intelligent agent systems using OpenClaw, privacy-respecting automation with Hermes, and a practical pair-programming workshop on Slimbook One mini PCs.
